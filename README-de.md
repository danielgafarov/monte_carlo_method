# Monte-Carlo-Simulation
[![en](https://img.shields.io/badge/lang-en-green.svg)](https://github.com/danielgafarov/monte_carlo_method)

Die **Monte-Carlo-Simulation** nutzt Zufallsstichproben, um numerische Lösungen für komplexe oder analytisch schwer lösbare Probleme zu finden. Dabei wird ein Szenario so häufig simuliert, bis sich die Wahrscheinlichkeiten der verschiedenen Ergebnisse über deren relative Häufigkeit zuverlässig ableiten lassen. 

In diesem Projekt wird die Methode angewandt, um die Gewinnwahrscheinlichkeiten verschiedener Lottovarianten präzise zu berechnen.
